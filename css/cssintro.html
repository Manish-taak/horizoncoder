Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a language used to describe the presentation of a document written in HTML. It controls the layout, colors, fonts, and overall visual appearance of the web page.

Types of CSS

1. Inline CSS

Inline CSS is applied directly within an HTML element using the style attribute. It is useful for quick, specific styling but not recommended for larger projects.

Example: This text is styled using inline CSS.

2. Internal CSS

Internal CSS is written inside the <style> tag within the <head> section of the HTML document. It is useful for styling a single page or when you want to keep styles specific to that page.

Example: The styles for this document (e.g., background color, font, etc.) are defined using internal CSS within the <head> section.

3. External CSS

External CSS is written in a separate file with a .css extension and linked to an HTML document using the <link> tag. It is the preferred method for styling large websites since it keeps the design separate from the content.

Example: You can create a file named styles.css and link it using the following line in the <head> section of your HTML:
<link rel="stylesheet" href="styles.css">
