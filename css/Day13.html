The clip-path CSS property allows you to define a clipping region to display a portion of an element. This is useful for creating interesting shapes for elements like images, divs, or even text. The clipped area will not be visible, and only the portion inside the path will be displayed.

clip-path Shapes Here are the most commonly used shapes that you can define with clip-path:

  • Circle (circle()): Defines a circular clipping area.
  • Ellipse (ellipse()): Defines an elliptical clipping area.
  • Polygon (polygon()): Defines a polygonal clipping area using a list of points.
  • Inset (inset()): Creates a rectangular clipping area with optional corner rounding.
  • Path (path()): Allows for custom shapes using SVG path syntax.
  1. circle(): Defines a circle, with syntax:
    • circle(radius at x-position y-position)
    • Example: circle(50% at 50% 50%)
      • Attributes
    • radius: Radius of the circle (e.g., 50% or 100px).
    • at x-position y-position: The position of the center of the circle. Defaults to the center of the element.
  2. ellipse(): Defines an ellipse, with syntax:
    • ellipse(rx ry at x-position y-position)
    • Example: ellipse(50% 30% at 50% 50%)
      • Attributes
    • rx: Horizontal radius.
    • ry: Vertical radius.
    • at x-position y-position: The position of the center of the ellipse.
  3. polygon(): Defines a polygon with multiple points:
    • polygon(x1 y1, x2 y2, ..., xn yn)
    • Example: polygon(50% 0%, 100% 100%, 0% 100%)
      • Attributes
    • List of points that define the polygon. Each point is defined by an x y pair representing percentages or lengths.
  4. inset(): Creates a rectangle, with optional corner rounding:
    • inset(top right bottom left round radius)
    • Example: inset(10% 20% 10% 20% round 20px)
      • Attributes
    • top, right, bottom, left: Define the distances from the edges.
    • round radius: Optional, to round the corners of the rectangle.
  5. path(): Defines a custom path using SVG path syntax.
    • Example: path('M10 10 H 90 V 90 H 10 Z')
      • Attributes
    • Use any valid SVG path string for custom shapes.
