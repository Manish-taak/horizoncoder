HTML Basic Interview Questions and Answers

1. What is HTML?
HTML (HyperText Markup Language) is the standard language used to create web pages. It uses a series of elements (tags) to structure content on the web, such as headings, paragraphs, images, links, and other types of media.
2. What is the purpose of the <!DOCTYPE> declaration?
The <!DOCTYPE> declaration defines the document type and version of HTML being used. It helps the browser render the page correctly by specifying the standards mode (e.g., HTML5).
3. What are HTML elements and tags?
HTML elements are individual components of an HTML document, such as a paragraph or a header. Tags are the markup that defines an element, usually consisting of an opening tag (like <p>) and a closing tag (like </p>).
4. What is the difference between block-level and inline elements?
Block-level elements occupy the full width of their container and start on a new line (e.g., <div>, <p>, <h1>). Inline elements occupy only as much width as their content requires and do not start on a new line (e.g., <span>, <a>, <strong>).
5. What is the <meta> tag used for?
The <meta> tag provides metadata about the HTML document, such as the character set, description, keywords, author, and viewport settings. Metadata is not displayed on the page but is used by browsers and search engines.
6. How do you create a hyperlink in HTML?
You create a hyperlink using the <a> tag with the href attribute specifying the URL of the page. Example:
<a href="https://www.example.com">Visit Example</a>
7. What is the difference between <id> and class attributes?
The id attribute is unique within a page and is used to identify a single element, while the class attribute can be used to group multiple elements together. id is typically used for JavaScript and CSS targeting, while class is often used for styling.
8. What are semantic HTML elements?
Semantic HTML elements clearly describe their meaning to both the browser and the developer. Examples include <header>, <footer>, <article>, and <section>.
9. How do you insert an image in HTML?
To insert an image, use the <img> tag with the src attribute for the image path and the alt attribute for alternative text. Example:
<img src="image.jpg" alt="Description of Image">
10. What is the purpose of the <form> element?
The <form> element is used to collect user input and send it to a server for processing. It can contain various input elements such as text fields, checkboxes, radio buttons, submit buttons, etc.
11. What is an iframe in HTML?
An <iframe> (inline frame) is used to embed another HTML page within the current page. It allows you to display content like a video, map, or another website.
12. What are data attributes in HTML?
Data attributes are custom attributes that start with data- and are used to store extra data on an HTML element. They are often used in JavaScript to process dynamic content.
13. What is the difference between <link> and <a> tags?
The <a> tag is used to create hyperlinks, while the <link> tag is used to define the relationship between the current document and an external resource, like a stylesheet.
14. What is the difference between HTML and XHTML?
HTML is the standard markup language for creating web pages, while XHTML (eXtensible Hypertext Markup Language) is a stricter, XML-based version of HTML. XHTML requires all elements to be properly closed and attributes to be in lowercase.
15. What is the role of the <script> tag?
The <script> tag is used to embed or reference JavaScript code within an HTML document. It allows you to add interactivity and dynamic content to web pages.
