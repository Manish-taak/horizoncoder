Understanding Table Elements

The table above demonstrates the use of various HTML tags to create and format a table:

  • <table>: Defines the table.
  • <caption>: Adds a caption/title to the table.
  • <colgroup> and <col>: Group and style table columns.
  • <thead>: Groups the header content in the table.
  • <tbody>: Groups the main body content in the table.
  • <tfoot>: Groups the footer content in the table.
  • <tr>: Defines a row in the table.
  • <th>: Defines a header cell in the table.
  • <td>: Defines a standard data cell in the table.
