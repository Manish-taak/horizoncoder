Examples of Text Formatting

Bold text: This text is bold using the <b> tag.

Important text: This text is important using the <strong> tag.

Italic text: This text is italicized using the <i> tag.

Emphasized text: This text is emphasized using the <em> tag.

Marked text: This text is highlighted using the <mark> tag.

Smaller text: This text is smaller using the <small> tag.

Deleted text: This text is marked as deleted using the <del> tag.

Water is represented by H2O using the <sub> tag for subscript text.

The 5th power is represented by x5 using the <sup> tag for superscript text.
