JavaScript Control Flow

1. Conditional Statements

Conditional statements are used to perform different actions based on different conditions.

If, Else If, and Else Statements

The if statement executes a block of code if a specified condition is true. The else if statement specifies a new condition to test, if the first condition is false. The else statement executes a block of code if all preceding conditions are false.

// Example of If, Else If, Else Statements
let age = 25;

if (age < 13) {
    console.log("You are a child.");
} else if (age < 20) {
    console.log("You are a teenager.");
} else {
    console.log("You are an adult.");
}

2. Loops

Loops allow you to run the same block of code a number of times.

For Loop

The for loop is used when you know how many times you want to execute a statement or a block of statements.

// Example of For Loop
for (let i = 1; i <= 5; i++) {
    console.log("Number:", i);
}

While Loop

The while loop executes a block of code as long as a specified condition is true.

// Example of While Loop
let count = 1;

while (count <= 5) {
    console.log("Count is:", count);
    count++;
}

Do-While Loop

The do-while loop is similar to the while loop, but it always executes the code block once before checking the condition.

// Example of Do-While Loop
let number = 1;

do {
    console.log("Number is:", number);
    number++;
} while (number <= 5);

3. Switch Statement

The switch statement is used to perform different actions based on different conditions. It's a cleaner way to write multiple if statements that compare the same expression with different values. 

// Example of Switch Statement
let fruit = 'apple';

switch (fruit) {
    case 'banana':
        console.log('Banana is $1');
        break;
    case 'apple':
        console.log('Apple is $2');
        break;
    case 'orange':
        console.log('Orange is $1.5');
        break;
    default:
        console.log('Sorry, we are out of ' + fruit);
}

Conclusion

Control flow statements in JavaScript allow you to determine how your program should execute code based on different conditions or repeatedly until a certain condition is met. Understanding these concepts is essential for building dynamic and responsive web applications.

JavaScript If-Else Statement Example

Question: How can you determine a student's grade based on their score using JavaScript if-else statements?

Answer:

Here is an example of how you can do that:

let score = 85;
let grade;

if (score >= 90) {
    grade = "A";
} else if (score >= 80) {
    grade = "B";
} else if (score >= 70) {
    grade = "C";
} else if (score >= 60) {
    grade = "D";
} else {
    grade = "F";
}

console.log("Alex's grade is: " + grade);

In this example, if the student's score is 85, they would receive a grade of "B".
