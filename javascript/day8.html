JavaScript BOM Methods: alert(), prompt(), confirm()

These methods are used to interact with the user. They are part of the window object in the Browser Object Model (BOM).

1. alert() Method

The alert() method displays an alert dialog with a specified message and an "OK" button.

                
function showAlert() {
    alert("Hello! This is an alert message.");
}

2. prompt() Method

The prompt() method displays a dialog with a text input field, allowing the user to enter some text. The input is then returned as a string.

                
function showPrompt() {
    let name = prompt("Please enter your name:");
    if (name !== null && name !== "") {
        alert("Hello, " + name + "!");
    }
}

3. confirm() Method

The confirm() method displays a dialog with a specified message, along with "OK" and "Cancel" buttons. It returns true if the user clicks "OK", and false if they click "Cancel".

                
function showConfirm() {
    let result = confirm("Do you want to proceed?");
    if (result) {
        alert("You clicked OK.");
    } else {
        alert("You clicked Cancel.");
    }
}
