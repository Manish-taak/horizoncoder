Day 2

JavaScript Operators and Expressions

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations.

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (Remainder)
  • ++ Increment
  • -- Decrement
// Example of Arithmetic Operators
let a = 10;
let b = 5;
console.log('a + b =', a + b);  // 15
console.log('a - b =', a - b);  // 5
console.log('a * b =', a * b);  // 50
console.log('a / b =', a / b);  // 2
console.log('a % b =', a % b);  // 0
console.log('++a =', ++a);      // 11
console.log('--b =', --b);      // 4

2. Assignment Operators

Assignment operators assign values to variables.

  • = Assign
  • += Add and assign
  • -= Subtract and assign
  • *= Multiply and assign
  • /= Divide and assign
  • %= Modulus and assign
// Example of Assignment Operators
let c = 10;
c += 5;   // c = c + 5
console.log('c += 5:', c);   // 15
c -= 3;   // c = c - 3
console.log('c -= 3:', c);   // 12
c *= 2;   // c = c * 2
console.log('c *= 2:', c);   // 24
c /= 4;   // c = c / 4
console.log('c /= 4:', c);   // 6
c %= 5;   // c = c % 5
console.log('c %= 5:', c);   // 1

3. Comparison Operators

Comparison operators are used to compare two values and return a boolean result.

  • == Equal to
  • === Strictly equal to (type and value)
  • != Not equal to
  • !== Strictly not equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to
// Example of Comparison Operators
let x = 5;
let y = '5';
console.log('x == y:', x == y);     // true
console.log('x === y:', x === y);   // false
console.log('x != y:', x != y);     // false
console.log('x !== y:', x !== y);   // true
console.log('x > 3:', x > 3);       // true
console.log('x < 3:', x < 3);       // false
console.log('x >= 5:', x >= 5);     // true
console.log('x <= 5:', x <= 5);     // true

4. Logical Operators

Logical operators are used to combine or invert boolean expressions.

  • && Logical AND
  • || Logical OR
  • ! Logical NOT
// Example of Logical Operators
let isTrue = true;
let isFalse = false;
console.log('isTrue && isFalse:', isTrue && isFalse); // false
console.log('isTrue || isFalse:', isTrue || isFalse); // true
console.log('!isTrue:', !isTrue);                     // false

5. Bitwise Operators

Bitwise operators work on the binary representation of numbers.

  • & Bitwise AND
  • | Bitwise OR
  • ^ Bitwise XOR
  • ~ Bitwise NOT
  • << Bitwise Left Shift
  • >> Bitwise Right Shift
  • >>> Bitwise Unsigned Right Shift
// Example of Bitwise Operators
let p = 5;   // 0101 in binary
let q = 3;   // 0011 in binary
console.log('p & q:', p & q);   // 1 (0001 in binary)
console.log('p | q:', p | q);   // 7 (0111 in binary)
console.log('p ^ q:', p ^ q);   // 6 (0110 in binary)
console.log('~p:', ~p);         // -6 (inverts bits)
console.log('p << 1:', p << 1); // 10 (1010 in binary)
console.log('p >> 1:', p >> 1); // 2 (0010 in binary)
console.log('p >>> 1:', p >>> 1); // 2 (0010 in binary)

6. Other Operators

JavaScript also has other operators for various purposes:

  • typeof - Returns the data type of a variable
  • instanceof - Checks if an object is an instance of a specific class or constructor
  • in - Checks if a property exists in an object
  • delete - Deletes a property from an object
  • ? Ternary Operator - Shortens conditional statements
// Example of Other Operators
let z = 42;
console.log('typeof z:', typeof z);                 // "number"
console.log('[] instanceof Array:', [] instanceof Array); // true
console.log('"length" in "Hello":', "length" in "Hello"); // true
let obj = { name: 'John' };
delete obj.name;
console.log('delete obj.name:', obj);               // {}
let result = z > 40 ? 'Greater than 40' : '40 or less';
console.log('Ternary Operator:', result);           // "Greater than 40"
